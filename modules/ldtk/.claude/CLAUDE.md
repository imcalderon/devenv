# LDtk Module

## Purpose
Installs the LDtk (Level Designer Toolkit) for 2D game level design.

## Key Files
| File | Purpose |
|------|---------|
| ldtk.sh | Main implementation |
| config.json | Module configuration (runlevel 4) |

## Interface
- `install` — Downloads and installs LDtk level editor
- `remove` — Removes LDtk installation
- `verify` — Checks LDtk is available
- `info` — Shows LDtk version

## Dependencies
None.
