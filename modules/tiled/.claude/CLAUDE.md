# Tiled Module

## Purpose
Installs the Tiled map editor for 2D game level design.

## Key Files
| File | Purpose |
|------|---------|
| tiled.sh | Main implementation |
| config.json | Module configuration (runlevel 4) |

## Interface
- `install` — Downloads and installs Tiled map editor
- `remove` — Removes Tiled installation
- `verify` — Checks Tiled binary is available
- `info` — Shows Tiled version

## Dependencies
None.
